If something's off

Where to look.

Skills not loading in Claude Code?

Confirm the skill folder lives at ~/.claude/skills/<name> and contains a SKILL.md. Restart Claude Code to refresh.

API key errors on a scored audit?

Scored audits need platform credentials (Google Ads, Meta, Klaviyo). Each skill's SKILL.md lists the environment variables it expects.

Python version issues?

Skills with Python helpers expect Python 3.11 or newer. Use pyenv or a recent system Python.

MCP server not connecting?

Check the MCP block in claude_desktop_config.json. Full walkthroughs live in the repo's docs/ directory.

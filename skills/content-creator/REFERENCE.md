# Content Creator — Reference

## Brand Voice Archetypes

| Archetype | Traits | Best For |
|-----------|--------|----------|
| The Expert | Authoritative, data-driven, precise | B2B, SaaS, professional services |
| The Friend | Conversational, warm, relatable | DTC, lifestyle, community brands |
| The Innovator | Forward-thinking, bold, disruptive | Tech startups, thought leadership |
| The Guide | Helpful, patient, educational | Education, how-to, documentation |
| The Rebel | Provocative, contrarian, direct | Challenger brands, opinion pieces |
| The Storyteller | Narrative-driven, emotional, vivid | Personal brands, case studies |

### Tone Matrix

| Dimension | Scale |
|-----------|-------|
| Formality | Casual ←→ Formal |
| Energy | Calm ←→ Energetic |

## Content Frameworks

### Blog Post Template

```markdown
# [Title with Primary Keyword] (50-60 chars)

Meta description: [155 chars with primary keyword, benefit, CTA]

## Introduction (100-150 words)
- Hook: question, statistic, or bold statement
- Context: why this matters now
- Promise: what the reader will learn

## [H2 with Secondary Keyword] (200-400 words)
- Key point with supporting evidence
- Example or case study
- Practical takeaway

## [H2 with Secondary Keyword] (200-400 words)
[Repeat pattern]

## [H2: Practical Application] (200-400 words)
- Step-by-step instructions
- Code snippets or templates
- Common pitfalls to avoid

## Conclusion (100-150 words)
- Summarize key takeaways (3 bullets)
- CTA: what to do next
- Internal link to related content
```

### LinkedIn Post Framework

```
[Hook — first 2 lines visible before "see more"] (max 210 chars)

[Story or insight — 3-5 short paragraphs]
[Use line breaks between each thought]
[One idea per paragraph]

[Key takeaway — bold or formatted differently]

[CTA — question or prompt for engagement]

[Hashtags — 3-5 relevant tags]
```

### Twitter/X Thread Framework

```
1/ [Hook tweet — must stand alone] (max 280 chars)

2/ [Context — why this matters]

3/ [Key insight #1]

4/ [Key insight #2 — with data if possible]

5/ [Counter-argument or nuance]

6/ [Practical takeaway]

7/ [CTA + link to full content]
```

### Newsletter Framework

```
Subject: [Curiosity gap or specific benefit] (40-60 chars)
Preview: [Extends subject, adds context] (40-90 chars)

## [Main Story]
- 300-500 words
- One key insight
- Personal angle or opinion

## [Quick Hits / Links] (3-5 items)
- [Title]: [One-line summary + link]

## [CTA]
- Reply prompt or resource link
```

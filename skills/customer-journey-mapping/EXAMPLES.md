# Customer Journey Mapping — Examples

## Example 1: B2B SaaS Full Journey Map

### Prompt
> Map the customer journey for our B2B analytics platform from first ad click to year-2 renewal. We sell to VP Marketing at mid-market companies.

### What the skill does
1. Maps 6 stages with touchpoints, emotions, pain points, and metrics for each
2. Identifies key transitions: ad click → demo request (12% conversion), demo → trial (45%), trial → paid (22%), year-1 → year-2 renewal (78%)
3. Highlights biggest drop-offs: trial → paid (complex onboarding), year-1 renewal (low feature adoption)
4. Produces actionable journey map document with improvement opportunities per stage

## Example 2: E-commerce Purchase Funnel

### Prompt
> Identify drop-off points in our purchase funnel. GA4 shows: homepage 100K → category 42K → product 18K → cart 4.2K → checkout 2.1K → purchase 1.4K.

### What the skill does
1. Calculates stage-to-stage conversion rates and identifies worst drop-off: category → product (57% drop)
2. Diagnoses likely causes for each major drop-off with qualitative indicators
3. Maps emotional journey alongside the funnel
4. Recommends fixes prioritized by revenue impact

## Example 3: Onboarding Journey Design

### Prompt
> Design an onboarding journey for our freemium project management tool. Most users sign up but never complete setup.

### What the skill does
1. Maps current onboarding: signup → empty dashboard (dead end)
2. Designs guided journey: signup → role selection → first project template → invite team → first task completion (activation moment)
3. Adds email support touchpoints at days 1, 3, 7
4. Defines activation metric: "created 3+ tasks with 1+ team member within 7 days"
